# 🩺 Pneumonia Detection using Deep Learning

## 📌 Overview
This project detects Pneumonia from Chest X-ray images using a pre-trained ResNet18 model with transfer learning.

## 🧠 Model Used
- ResNet18 (pretrained on ImageNet)
- Fine-tuned final classification layer

## 📊 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- AUC-ROC

## 📂 Dataset
Chest X-ray dataset with two classes:
- NORMAL
- PNEUMONIA

Dataset not included in this repository.
